>git add -A[to adding all untracked files to staging area]
>git commit -m"commit messege"[to commit added file to local repository]
>git clone[to take exact copy from remote repository to local worksapce]
>git pull[to take chnges from remote repository to local worksapce]

>git push remote_name branch_name
